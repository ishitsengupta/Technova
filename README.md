# TechNova Solutions - Corporate Website

**Author:** Ishit Sengupta  
**Project Context:** Junior Web Developer Training Project @ Aurika Infotech    

---

## Project Description
This project is a fully responsive, 5-page corporate website built for a fictional client, **TechNova Solutions** (Tagline: "Innovation at Every Step"). The website serves as a digital storefront for an IT Services, Software Development, and Corporate Training company. It features a modern, minimalist design system, seamless navigation, and interactive elements—all built entirely from scratch without the use of external CSS frameworks or JavaScript libraries.

## Purpose of the Website
The primary objective of this project was to design and develop a production-ready client deliverable that demonstrates proficiency in core web technologies (HTML5, CSS3, Vanilla JS). 

Business goals for the website include:
- Establishing a professional brand identity through a minimalist, high-contrast aesthetic.
- Clearly communicating technical services and pricing tiers.
- Showcasing past work and case studies to build client trust.
- Providing a functional, validated touchpoint for lead generation (Contact Form).

## Source of Data & Content
*Note for evaluation:* Since TechNova Solutions is a fictional entity, the textual content, service descriptions, and case studies were synthesized by analyzing real-world B2B technology consulting firms (such as ThoughtWorks, EPAM Systems, and Infosys). The organizational timeline, pricing structures, and engineering philosophies were adapted to fit the specific constraints of the Aurika Infotech training brief. Visual assets are sourced dynamically via Unsplash source URLs.

## Tech Stack
- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Custom Properties/Variables, CSS Grid, Flexbox)
- **Interactivity:** Vanilla JavaScript (ES6)
- **Architecture:** Component-based CSS methodology with a single shared stylesheet.
